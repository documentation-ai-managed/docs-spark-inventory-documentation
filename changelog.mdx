## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Enhanced AI predictions with improved accuracy for seasonal products
- Added bulk import functionality for large inventories
- Introduced advanced reporting dashboard with customizable metrics

## Improvements

- Faster webhook delivery with optimized queuing system
- Improved error messages for API authentication failures
- Enhanced mobile responsiveness in the dashboard

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Complete API redesign with RESTful endpoints
- Real-time inventory synchronization across multiple locations
- Machine learning model updates every 24 hours

## Breaking Changes

- API authentication now requires Bearer token format
- Webhook payload structure has been updated
- Deprecated legacy SDK versions below 1.0

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Initial webhook support for inventory events
- Added support for multiple product categories
- Introduced basic AI demand forecasting

## Bug Fixes

- Fixed calculation errors in cash flow projections
- Resolved issues with duplicate product entries
- Corrected timezone handling in reports

</Update>
